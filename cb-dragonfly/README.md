#### 1. 실행 환경

- ##### 공식환경
  - OS: Ubuntu 18.04
  - Container: Docker 19.03
  - Build: Go 1.16
- ##### 시험환경
  - OS: Ubuntu 18.04
  - Container: latest Docker
